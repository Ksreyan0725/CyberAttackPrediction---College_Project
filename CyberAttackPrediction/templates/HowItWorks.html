Session Security & Persistence

How "Remember Me" Works

When you check "Remember Me", the system securely caches an encrypted reference to your session in your local browser storage. This allows the Speed Login feature to identify your machine and grant access without re-entering credentials.

Security Tip: Only use this feature on your personal, private computer. Never use "Remember Me" on public or shared workstations.

Smart Logout Logic

Your security is our priority. CyberShield AI implements a dual-layer logout system:

  • Inactivity Guard: Automatically logs out after 1 hour of idle time to prevent unauthorized physical access.
  • Context-Aware Protection: If you refresh the Results Page without "Remember Me" active, the system clears the session to protect sensitive prediction data.
Under The Hood

1. The Database (users.db): Think of this as the secure filing cabinet for the application. When you sign up, your account details and passwords are securely saved into a local file named users.db. Whenever you try to log in, the system checks this cabinet to make sure you belong.

2. The Server Memory (Flask Session): Once you log in successfully, the server gives your browser a special, temporary "VIP wristband" (known as a secure session cookie). As long as you wear this wristband, you can freely use protected tools like the AI Predictor without having to type your password again.

3. The Browser Memory (LocalStorage): Standard login forgets you once you close the tab. If you click "Remember Me", the application commands your web browser (like Chrome or Edge) to memorize your email. The very next time you visit the website, your browser uses this memory to automatically fill everything in for you like magic!
