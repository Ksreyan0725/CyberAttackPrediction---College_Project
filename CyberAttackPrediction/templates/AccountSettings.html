Tactical Operations Guide

Upload your network snapshot (CSV) in the **Analysis Engine**. The Random Forest algorithm performs deep-packet inspection and classifies the signature. GenAI will then generate a tactical report for any detected threats.

As a Master Admin, navigate to the **User Registry** to monitor active system occupants.

• Override Protocol: Use 'Reset Key' to manually set new credentials for any user account.
• Termination Protocol: Use 'Terminate' to permanently purge user data from the central mainframe.
All administrative actions are logged and take effect instantly across all nodes.

The **Status Monitor** provides a real-time heartbeat of the CyberShield AI. Monitor model latency, detection accuracy, and total blocked threats. In high-glare environments, use the **Environmental Themes** to switch between OLED Black and Cyber Green modes.

After each classification (Neptune, Satan, Back, etc.), the system automatically invokes the **GenAI Layer**. This layer analyzes the packet signature and generates a human-readable tactical response, including mitigation steps and risk assessment.

For manual configuration without using the dashboard, you can direct-edit the project files at the following locations:

• User Registry: Project/CyberAttackPrediction/users.json
Manual Edit: Open this file to see usernames and hashed passwords. You can manually delete an entry to purge a user.

• Training Datasets: Project/CyberAttackPrediction/Dataset/
Management: Drop new CSV files here to update the training pool. Ensure headers match the original schema for model training.

• AI Model Binaries: Project/CyberAttackPrediction/model/
Observation: This directory contains the serialized Random Forest model and the data scaler. Do not edit these files manually.

• Runtime Uploads: Project/CyberAttackPrediction/Dataset/uploaded_test.csv
Inspection: Every time an analysis is run, the uploaded data is mirrored here for temporary processing.

• UI Templates: Project/CyberAttackPrediction/templates/
Customization: All HTML structures for the login, dashboard, and analysis pages are stored here.

• Static Assets: Project/CyberAttackPrediction/static/
Styling: Contains the images and CSS responsible for the premium visual effects of the Command Center.
